# @jonny/touchable-scale

Button that animates scale when pressed

## Installation

```sh
npm install @jonny/touchable-scale
```

## Usage

```js
import TouchableScale from "@jonny/touchable-scale";

// ...

const result = await TouchableScale.multiply(3, 7);
```
